# v1.1.16

**Maintenance release. Nothing changes about how Quill behaves** — safe to update, nothing to relearn.

## What changed

Internal cleanup and build-reliability work only:

- Removed unused dependencies and dead code (~500 lines, 11 packages), shrinking the app's
  install footprint and supply-chain surface. This includes legacy file-access code paths that were
  no longer used, with the security confinement on the live paths verified unchanged.
- Made the automated test suite's coverage checks deterministic, so the build pipeline stops
  failing intermittently on unchanged code.

Nothing in the editor, your documents, comments, track-changes, or the Claude integration changes.

Update the same way as always: quit Quill, then drag `Quill.app` from the synced Drive folder into
`/Applications`, replacing the old copy. Documents and settings carry over.
