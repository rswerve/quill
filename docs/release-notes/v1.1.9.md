# Quill v1.1.9

**Claude's edits now tell you the truth.**

This release fixes five defects in the pipeline that turns Claude's replies into tracked suggestions — all variations on one theme: the app claiming an edit succeeded (or failed for the wrong reason) while something else actually happened.

- **Formatting requests no longer bounce as "malformed."** When Claude describes a formatting change redundantly (repeating the found text in the replacement), Quill now recognizes the intent and applies it as a clean formatting suggestion.
- **Multi-paragraph matches work.** Claude sometimes copies blank-line separators from the Markdown source when quoting several paragraphs; Quill now matches those quotes instead of reporting "this text isn't in the document" while the text sits on screen.
- **Structural changes fail honestly.** Merging or splitting paragraphs, bullets, or headings can't yet be represented as a tracked suggestion — previously such edits were silently discarded while the reply claimed success. Quill now says so plainly and suggests making the change in Editing mode.
- **Code blocks are protected.** A Claude text edit inside a code block previously _duplicated_ the content untracked; a formatting edit there silently did nothing while reporting success. Both are now refused with an honest notice.

No file-format or data changes. Documents, comments, suggestions, chat threads, and linked sessions carry over unchanged.

## Install

Inside Truss, Quill is shared as a ready-to-run app on the team's Google Drive — copy `Quill.app` from the shared drive (in Finder, from your synced folder) to your Applications folder and open it. The [README](../../README.md#installing-quill) has the details and the [User Guide](../USER_GUIDE.md) a full walkthrough. The Claude features require the [Claude Code](https://claude.com/claude-code) CLI on the same Mac.

## Known limitations

- macOS only. Not yet Apple-notarized — a copy downloaded through a browser (e.g. the Drive website) is typically quarantined and trips Gatekeeper on first launch; open the Finder-synced copy instead, or use **System Settings → Privacy & Security → Open Anyway**.
- Single-user: multiple documents open as tabs in one window; no multi-user collaboration or cloud sync.
- Suggesting mode cannot yet merge, split, or restructure blocks as a tracked change — Quill will tell you when a request needs Editing mode.
